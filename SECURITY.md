# Security Policy

## Supported Versions

The latest published `0.x` release receives security fixes.

## Reporting a Vulnerability

Please report security issues **privately** — do not open a public issue.

- Preferred: use GitHub's [**Report a vulnerability**](https://github.com/DanMat/query-server/security/advisories/new)
  (Security tab → Advisories), or
- Email **dannymatthew@gmail.com** with details and, if possible, a reproduction.

You can expect an initial response within a few days. Once a fix is available,
a patched release will be published and the advisory disclosed.

Thank you for helping keep this project and its users safe.
